# form-admin

Next.js 16 App Router admin for building and managing forms.

## Purpose

Internal admin UI for creating, editing, and publishing dynamic forms. Includes a drag-and-drop builder (Wave 3 / L7), conditional logic editor, theme editor, submission viewer, webhook configuration, and RBAC management.

## Port

`4201` — run locally with `pnpm dev`.
